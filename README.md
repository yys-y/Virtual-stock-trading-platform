# 虚拟股票交易平台
![story](https://github.com/WHUCScoke/Virtual-stock-trading-platform/blob/master/picture/process.png)
## 项目简介
这是一个基于qt的虚拟股票交易平台实现，可以提供实时股票数据更新、交易、资产管理等功能。
## 应用技术
文件I/O操作多线程处理, Qt界面开发,sqlite数据库处理
## 实现功能
1. **用户账户创建和管理功能**   
2. **虚拟货币充值和提现功能**   
3. **股票买入和卖出功能**   
4. **交易历史记录和报表功能**   
5. 实时股票行情显示功能
6. 用户投资组合管理功能
7. 用户风险评估和提示功能
8. 股票市场新闻和分析功能
9. 社交功能，如论坛和交流区
10. 模拟交易比赛和排行榜功能
